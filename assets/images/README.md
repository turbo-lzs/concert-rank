## 📝 注意事项

1. 支持的格式：JPG、PNG
2. 建议图片大小：每张图片不超过 500KB
3. 文件名必须是英文，不要包含空格或特殊字符
4. 放置图片文件后，刷新相关页面即可看到效果

## 🔧 使用方法

1. 将你的图片文件复制到此文件夹
2. 确保文件名与上方列表一致
3. 访问 `image-manager.html` 页面查看图片状态
4. 点击"更新数据库图片路径"按钮更新数据库
5. 刷新投票页面查看效果

## 🌐 管理页面

访问以下页面进行图片管理：
- 本地开发：`http://localhost:3001/image-manager.html`
- 部署后：`https://你的域名/image-manager.html`
